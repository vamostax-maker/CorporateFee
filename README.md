# 세무법인 엑스퍼트 성수점 - 25년 법인세 조정료 관리 시스템

## 주요 기능

1. **담당자별 대시보드** - 6명 담당자별 조정료 집계, 입금/미입금 현황, 수금률 진행바
2. **전체 집계 카드** - 총 청구액, 입금 완료, 미입금, 수금률 한눈에 확인
3. **입금 체크** - 업체별 입금 체크박스 (체크 시 초록색 강조)
4. **실시간 합계** - 체크된 금액 합계 실시간 집계
5. **필터링** - 담당자별, 입금상태(전체/입금/미입금)별 필터
6. **검색** - 업체명, 대표자, 사업자번호 검색
7. **집계표** - 조회된 전체 합계 / 입금 합계 / 미입금 합계 별도 표시

## 표시 컬럼 (4개)
- 24년 귀속 조정료
- 25년 귀속 조정료 [보수표 기본]
- 26년 조정료 [공급가]
- ★ 26년 조정료 [최종 VAT] ← 메인 집계 기준
